# Техническое задание — Лендинг ВИЗИ ПРО

## Общие требования

- **Mobile-first** — основной трафик с телефонов
- **Стиль:** минималистичный, тёмный фон, акцентный цвет — оранжевый
- **Шрифт:** современный, без засечек
- **Структура:** 11 блоков

### Цветовая схема
- Фон: тёмный (#1a1a1a или близкий)
- Акцент: оранжевый (как на кнопках)
- Текст: белый

### Адаптивность
- Mobile-first
- На десктопе карточки можно показывать в ряд

### Анимации
- Слайдер до/после — интерактивный ползунок
- Карусель — свайп на мобайле
- FAQ — аккордеон с плавным раскрытием

## Блок 1: Первый экран

### Элементы:
- Лого ВИЗИ ПРО (левый верхний угол)
- Кнопка "Войти" (правый верхний угол)
- Картинка: сгенерировать изображение — замерщик показывает клиенту телефон с визуализацией потолка, клиент улыбается. Стиль как у имеющихся фото, но другой ракурс (не дублировать картинки из карусели).

### Текст:

```
Приложение для потолочников

Помогает заключать договор прямо на замере.
Покажи клиенту, как будет выглядеть его потолок — до монтажа.

Устал слышать «я подумаю»?
Это можно исправить.
```

```
[Попробовать бесплатно]
```

## Блок 2: Как это работает

### Статус: Готов

Карусель с 3-4 слайдами. Оставить как есть.

### Слайды:
1. Сфотографируй комнату
2. Выбери потолок и освещение
3. Покажи клиенту результат
4. (опционально) Получи договор

### Элементы:
- Горизонтальный слайдер с картинками процесса
- Точки навигации под слайдером
- Подпись: "Создай визуализацию за 30 сек"

## Блок 10: FAQ

### Заголовок:
```
Частые вопросы
```

### Формат: аккордеон (нажимаешь — разворачивается ответ)

### Вопросы и ответы:

**1. Сколько времени занимает визуализация?**
```
Одна визуализация — около 30 секунд.
```

**2. Сложно ли разобраться в приложении?**
```
Приложение интуитивно понятное. Ставка сделана на простоту и скорость.
```

**3. Нужно ли обучение?**
```
Внутри есть обучение: как продавать с помощью приложения, как располагать освещение в разных зонах и как сочетать разные типы решений. Обучение от дизайнера.
```

**4. Работает на iPhone и Android?**
```
Да. Ещё можно на планшетах и на ПК.
```

**5. Можно ли показать несколько вариантов клиенту?**
```
Да, можно создать несколько вариантов и показать клиенту на выбор.
```

**6. Как быстро окупается подписка?**
```
С одного заказа можно купить подписку на год.
```

**7. Что если закончились генерации на день?**
```
Можно докупить дополнительные генерации на день.
```

**8. Насколько реалистично выглядит результат?**
```
Максимально приближено к реальному виду после монтажа.
```

**9. Какие варианты потолков поддерживаются?**
```
Парящие, теневые, обычные примыкания. Несколько видов точечных светильников, несколько видов люстр, трековые системы, световые линии, карнизы с подсветкой и без. Постоянно дорабатываем и добавляем новые варианты.
```
